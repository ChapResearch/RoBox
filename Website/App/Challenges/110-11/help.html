Challenge #11

Light Up Lazer Run!

In this challenge you will program your robot to Light Up all 3 LED lights and turn off one after another when it is shot with an IR blast. After all the lights turn off, your robot should stop moving.

You will be using the LED block to turn on and off you're robots lights.

HIny: Use an If block inside a Repeat Forever block to make sure you're robot is lways checking if it is hit. Good Luck!
