Challenge #7

The 3 Lines!

In this challenge you will be coding your robot to go forward until it sees 3 lines. It sounds easy at first, but you are going to have to use loops and your light sensor really well to make it work successfully!

Remember, you already mastered how to do this with one line, so see if you can do it 3 times in the same program!

Try using loops and what you learned when stopping at a line in Challenge 5 to solve this problem.
