Challenge #6

Escape!

In this challenge you will program your robot to Escape from inbetween 3 blocks!

You are going to be using a Repeat block again to complete this challenge. Try using what you learned while using the Ultrasonic sensor. That way your robot can check each side of it before driving out of the blocks.


When you are finished your robot will be placed inbetween the 3 blocks in a random position. So make sure it uses sensors to complete this challenge. Good Luck!
