Challenge #8

Lazer Run!

In this challenge you will program your robot to Light Up and make a sound when its hit by an Infared (IR) Lazer!

Now you have 2 new blocks, the LED block, and the Beep block.

The LED block can turn onand off one of the three lights on the robot. The following blocks turn on the first set of lights then turns them off a second later.


The Beep block makes the robot make a sound for a certain number of seconds. The robot can make a low sound, a high sound, and a middle sound. The following Beep blocks plays a low sound for 2 seconds.


Remember to use an If block to check if you've been hit, and use a Repeat Forever block so your robot is always checking if it's it. Good Luck!
