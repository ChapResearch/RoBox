Challenge #6

Light Up Lazer Fight!!!

In this challenge you program your robot to Light up and make a sound when its hit by an Infared (IR) lazer!

Now you have 2 new blocks, the LED block, and the Beep block.

The LED block turns on one of the 3 sets of lights on the robot, but it doesnt turn them off so don't forget to use another LED block to turn them off when you're done. The following blocks turn on the first set of lights then turns them off a second later.


The Beep block makes the robot make a sound at a certain frequency for a certain number of seconds. The following Beep blocks plays a sound at a frequency of 260Hz for 2 seconds.


Remember to use an if block and a Repeat Forever block. Good Luck!
