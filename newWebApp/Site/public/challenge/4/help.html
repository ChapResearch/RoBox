Challenge #4

Drive To The Wall!

In this challenge you program your robot to stop right before it slams into a wall! But how do you do that?

Fortunately, the RoBox has an Ultrasonic sensor on the front! This sensor sends out a beep of sound and listens for the sound to return. It can tell how far it is away from the wall by measuring how long the sound takes to get back to the sensor after the beep.

You will be using the Ultrasonic sensor block inside a Repeat Until block. A Repeat Until block repeats a set of blocks until the condition is met. The following would repeat a set of blocks until the Ultrasonic sensor sees an object 20 millimeters (mm) away.


Don't try to listen for the beep...it is much too high for you to hear.
