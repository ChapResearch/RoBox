Challenge #10

Light Up Lazer Run!

In this challenge you will program your robot to Light Up all 3 LED lights and turn off one after another once you are shot. After all the lights turn off, your robot should stop moving.

You will be using the LED block and the Beep block.

The LED block turns on one of the 3 lights on the robot, but it doesnt turn them off. So don't forget to use another LED block to turn them off when you're done! The following blocks turn on the first set of lights then turns them off a second later.


Try to use an If block and a Repeat Forever block to complete the challenge. Good Luck!
